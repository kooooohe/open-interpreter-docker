# open-interpreter-docker
open-interpreter docker environment

### Docker

#### Docker build

```sh
docker compose build --pull
```

#### Run docker

```sh
docker compose up
```

#### Run app

put your api key on .env
`OPENAI_API_KEY=YOURKEY`
```sh
root@hostname:~# interpreter
```
